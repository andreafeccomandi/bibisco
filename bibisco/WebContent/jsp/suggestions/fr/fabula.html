5. Définissez le récit

Le récit est une narration d'événements ordonnés selon une séquence logico-temporelle.

Tout d'abord, vous devez définir brièvement le récit, indépendamment de l'intrigue, qui est l'ordre dans lequel les événements sont présentés (narration linéaire, retour en arrière, anticipation).

Une fois que vous avez entièrement défini le récit, il est plus facile pour vous de déterminer la manière la plus efficace pour développer l'intrigue.
